# Getting Started

This repository is built for quick practical use.

## Suggested first path

- Read the repository README.
- Browse `docs/Home.md`.
- Pick one quick resource:
  - `docs/getting-started-with-wandb.md`
  - `docs/getting-started-with-dvc.md`
  - `references/reproducibility-checklist.md`

## For new contributors

1. Read `CONTRIBUTING.md`.
2. Copy a suitable template from `templates/`.
3. Keep your first contribution small and useful.
4. Open a pull request using the PR template.

## Versioning and updates

- Treat pages as living documents.
- Add or update a **Last updated** line on substantial guides.
- If a guide is outdated, open an issue and mark the page clearly until refreshed.
