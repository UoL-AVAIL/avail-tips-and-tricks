# Standards for Tutorials and Notebooks

## Goal

Help someone complete a useful task quickly and reproducibly.

## Required sections

- Purpose
- Prerequisites
- Step-by-step instructions
- Common pitfalls
- Further reading

For notebooks, also include:
- environment details;
- how to run;
- expected outputs.

## Practical standards

- Keep examples small and realistic.
- Prefer one clear workflow over many options.
- Include file paths and commands exactly.
