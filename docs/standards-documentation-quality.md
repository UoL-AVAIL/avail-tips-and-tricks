# Standards for Documentation Quality

## Writing style

- Use clear British English.
- Keep paragraphs short and practical.
- Prefer plain language over jargon.

## Structure

- Start with purpose and audience.
- Provide exact steps.
- Include expected output or success criteria.
- Link related pages.

## Maintenance

- Mark templates clearly as templates.
- Add **Last updated** to substantial pages.
- Raise an issue for outdated content.
