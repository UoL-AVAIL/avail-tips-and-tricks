# Shared Practices

Shared practices are common tools, techniques, and working patterns that improve research quality across the group.

## Why this matters

Good shared practice helps us:
- make work reproducible;
- reduce avoidable mistakes;
- speed up onboarding;
- communicate methods clearly.

## Focus areas

- experiment tracking
- versioning data and code
- reproducibility checklists
- practical writing workflows
- lightweight automation
