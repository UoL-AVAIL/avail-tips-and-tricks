## Tools

### Programming / IDEs / AI-assisted coding
- VS Code: https://code.visualstudio.com/
    - A free, open-source IDE with a rich ecosystem of extensions.
    - Includes support for GitHub Copilot, Codex, Claude Code, etc.
- Jupyter Notebook: https://jupyter.org/
    - An open-source web application for creating and sharing documents that contain live code, equations, visualizations, and narrative text.
    - Supports various programming languages through kernels, including Python, R, and Julia.
- GitHub Copilot: https://github.com/features/copilot
    - AI-powered code completion and suggestions, with various coding agents available.
    - Free for students and educators, with a paid plan for professionals.
    - Register as a student or educator here: https://github.com/education/students
- Codex: https://chatgpt.com/codex
    - OpenAI's code generation model, integrated into various IDEs and platforms.
    - Available as a VS Code extension and through the OpenAI API.
- Claude Code: https://code.claude.com/
    - Anthropic's AI coding assistant, designed to help with code generation and debugging.
    - Available as a VS Code extension and through the Anthropic API.
- Cursor: https://www.cursor.so/
    - AI-powered code completion and suggestions, with a focus on productivity and collaboration.
    - Essentially a modified version of VS Code with AI features built-in.
    - Eligible university students get one free year of Cursor Pro.
- Google Antigravity: https://antigravity.dev/
    - Another IDE and CLI for and agentic, AI-assisted coding.
    - Free for individuals but with weekly rate limits.

### Academic writing
- LaTeX: https://www.latex-project.org/
    - A high-quality typesetting system for producing scientific and technical documents.
    - Free to use, with various editors and distributions available.
- Overleaf: https://www.overleaf.com/
    - An online LaTeX editor for collaborative writing and publishing.
    - Free and paid plans available, with a web app and desktop app.
- Prism: https://prism.openai.com/
    - Overleaf-like collaborative writing in LaTeX with LLM assistance.
    - Developed by OpenAI, free to use.
- Markdown: https://www.markdownguide.org/
    - A lightweight markup language for creating formatted text using a plain-text editor.
    - Free to use, with various online editors and converters available.
    - Can be used in conjunction with pandoc for document preparation: https://github.com/UoL-AVAIL/pandoc-writing-template
- Typst: https://typst.app/
    - A modern typesetting system for creating high-quality documents with a focus on simplicity and ease of use.
    - Free to use, with a web app and desktop app available.
- Quarto: https://quarto.org/
    - An open-source scientific and technical publishing system built on Pandoc, supporting multiple output formats.
    - Free to use, with a web app and desktop app available.
- Notion: https://www.notion.so/
    - An all-in-one workspace for note-taking, project management, and collaboration.
    - Free and paid plans available, with a web app, desktop app, and mobile app.
- Obsidian: https://obsidian.md/
    - A knowledge management and note-taking app that supports linking notes and creating a personal knowledge base.
    - Free and paid plans available, with a desktop app and mobile app.
- reMarkable: https://remarkable.com/
    - A digital paper tablet for reading, writing, and sketching, with support for PDF and ePub formats.
    - Paid device with optional cloud storage and collaboration features.

    - Free to use with a Google account.
- Speechify: https://speechify.com/

    - Free and paid plans available, with a browser extension for easy access.
- Rayyan: https://rayyan.ai/

    - Free to use with optional paid features for advanced functionality.

- Zotero: https://www.zotero.org/
    - A free, open-source reference management tool for organising and citing research sources.
    - Available as a desktop app, browser extension, and web app.

## Productivity tips and workflow suggestions

- Pomodoro technique: Use a timer to break work into intervals, traditionally 25 minutes in length, separated by short breaks.
- "If it will take <5 minutes, do it now." - A productivity tip to avoid procrastination and keep tasks manageable.
- Create weekly slides to summarise progress, serving as a kind of "lab book" you can refer back to when writing papers or preparing presentations.
- Maintain a spreadsheet of paper summaries as useful reminders of things you've read and to help with literature reviews.
- Have a master "to-do list"; use GitHub issues, Trello or Teams to keep track of tasks and deadlines.
- Treat yourself everyone once in a while; take a break, go for a walk, or do something enjoyable to recharge your energy and maintain motivation.
- Have a proper lunch break; step away from your desk, eat mindfully, and give your brain a chance to rest and recharge.
- Journalling or blogging about your work can help you reflect on your progress, clarify your thoughts, and share your insights with others.
- Physical activity is important for both body and brain; regular exercise can improve focus, memory, and overall well-being.
