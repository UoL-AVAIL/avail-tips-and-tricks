# Contribution Guide

This page summarises how AVAIL members contribute practical, reusable resources.

- Process details: `CONTRIBUTING.md`
- Tutorials/notebooks standards: `standards-tutorials-notebooks.md`
- Script standards: `standards-scripts.md`
- Documentation quality standards: `standards-documentation-quality.md`

## Preferred contribution style

- Start small.
- Show a real use case.
- Include exact steps and expected outcome.
- Keep wording direct and friendly.
