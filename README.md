# AVAIL Tips and Tricks

A shared knowledge base for the AVAIL group (students and staff) to capture practical research workflows, tooling guides, and reusable examples.

## What is AVAIL?

AVAIL is a research-focused group that shares methods, tools, and working practices across projects. This repository is our living handbook: practical, concise, and easy to reuse.

## What this repository is for

Use this repository to:
- capture what we learned at the AVAIL (Not Far) Away Day;
- publish tutorials and walkthroughs from AVAIL members;
- share notebooks, scripts, and lightweight examples;
- provide “getting started with X” guides for common tools;
- maintain reusable resources for new members.

## How to contribute

Please read [`CONTRIBUTING.md`](CONTRIBUTING.md).

In short:
1. choose a small, practical contribution;
2. use the templates in `templates/`;
3. keep examples reproducible and concise;
4. open a pull request with a clear summary and expected outcome.

## Living resource guidance

This is a living repository, not a static archive.

- Prefer short, actionable pages over long theory-heavy notes.
- Update pages when tools or workflows change.
- Add a brief **Last updated** line to substantial guides.
- Link related pages so new members can find useful content in under a minute.
